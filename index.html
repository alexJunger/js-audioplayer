Importing MP3s

You can import mp3-files by clicking the "open" icon beneath the tracklist. Please note however, that mp3-files will not be restored when reloading, at this time.

Importing JSON

You can import JSON-files by clicking the "open" icon beneath the tracklist. Those tracks imported from JSON will be restored when you reload the player. The structure of the JSON should look like this:

{
 "tracks": [
  {
   "url": "assets/filename.mp3",
   "title": "Tracktitle",
   "artist": "The Artist",
   "genre": "The Genre",
   "album": "The Album",
   "year": "2014"
  },
  {
  ...
  }
 ]
}
